## Summary

| Platform | Strengths | Gaps |
|---|---|---|
| **SAKAI** | Full-stack AI, real-time demand, driver tools, safety layer, continuous learning | New entrant; requires user adoption and live data to improve over time |
| **JTransit** | Live vehicle tracking, route search | No AI, no driver tools, no safety features, no fare estimation |
| **Sakay.ph** | Route search, fare estimation | No real-time data, no AI, no driver support |
| **Komyut** | Route search, fare estimation | No real-time data, no AI, no driver support |
| **Google Maps** | Route alternatives, fare estimation, offline support, behavioral learning | Not PUV-specific; no driver mode, no demand visibility, no safety layer |

> SAKAI is the only platform in this comparison purpose-built for the Philippine PUV ecosystem — addressing both the commuter experience and the driver's operational needs through a unified, AI-driven system.
